# Rakhat Portfolio

Готовый проект под Vercel на Next.js.

## Как запустить локально

```bash
npm install
npm run dev
```

## Как загрузить на Vercel

1. Зайди на Vercel.
2. Нажми **Add New Project**.
3. Выбери папку с этим проектом или загрузи её в GitHub.
4. Нажми **Deploy**.

Vercel сам определит Next.js и соберёт сайт.
